# WTF-CSS
WTF CSS教程，总结/搬运自MDN CSS教程，帮助新人快速入门CSS，并写出Uniswap页面和NFT铸造页面。

---

## 开发工具

我们建议大家使用 [VSCode](https://code.visualstudio.com/download) 开发 HTML 和 CSS，并用浏览器（Chrome）打开`.html`文件进行浏览。
